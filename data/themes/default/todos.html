Todo items

Following fortylines organizational principles, features, bugs and issues are tracked as todo items with each item being its own text file living along side the other files in the source repository. The web presentation engine provides an easy and convenient way to create, browse and comment on todo items but nothing prevents any contributors to manipulate todo items from the command line or with a text editor as it would be done with any other file in the source repository.
